# saas-pricing-page
Responsive HTML/CSS pricing page designed to present plans, compare features, highlight value, and guide visitors toward a trial, demo, or purchase decision.

## Built with

- HTML5
- CSS3
- Responsive design
- Semantic structure
- Vanilla front-end workflow

## Project purpose

This project was created as a portfolio piece to demonstrate how a SaaS pricing page can be structured to help visitors compare options clearly and choose the plan that fits their needs.

## Key sections

- Hero section
- Billing toggle
- Pricing cards
- Featured plan highlight
- Feature comparison table
- Plan recommendation section
- Trust / security section
- FAQ
- Final conversion CTA

## Portfolio focus

This project demonstrates front-end web production, pricing layout structure, plan comparison, responsive design, visual hierarchy, and conversion-focused calls to action.
